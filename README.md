## Conclusion

Next Auth is awesome if you want your users to only use external login ways, like sign in with Google etc.
But if you want to allow the most basic way of signing in, via address email, it sucks xD. I didn't test it to the fullest, but asked Ai to tell me if it's worth it and it said that, it doesn't have script attacks protection on this method. Technically you can add them yourself, but if you do that you should just build the entire login system by yourself.
